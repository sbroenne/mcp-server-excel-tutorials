## üöÄ Prerequisites

- **Windows 10/11** with Excel installed (Microsoft 365 or Excel 2016+)
- An AI assistant that supports MCP:
  - [VS Code with GitHub Copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot)
  - [Claude Desktop](https://claude.ai/download)
  - [Cursor](https://cursor.sh/)
  - [Windsurf](https://www.codeium.com/windsurf)

## üì¶ Installation

### Option 1: VS Code Extension (Recommended)
1. Install the [Excel MCP Server extension](https://marketplace.visualstudio.com/items?itemName=sbroenne.excel-mcp-server)
2. Open VS Code and start chatting with Copilot about Excel!

### Option 2: Manual Setup
See the [Installation Guide](https://github.com/sbroenne/mcp-server-excel/blob/main/docs/INSTALLATION.md) for detailed instructions.
