### Create a Workbook
```
Create a new Excel file at C:\Temp\my-first-workbook.xlsx
```

### Add Data
```
Open C:\Temp\my-first-workbook.xlsx and add:
- Row 1: Name, Age, City (headers)
- Row 2: Alice, 28, New York
- Row 3: Bob, 35, Los Angeles
Then save and close.
```

### Read Data
```
Read the contents of C:\Temp\my-first-workbook.xlsx from range A1:C3
```

### Add a Formula
```
Open the workbook, add "Average" in A5 and =AVERAGE(B2:B3) in B5, save and close.
```
