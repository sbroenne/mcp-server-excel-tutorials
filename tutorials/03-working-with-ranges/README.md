## ‚ö° Quick Reference

| Operation | Description |
|-----------|-------------|
| `get-values` | Read data from cells |
| `set-values` | Write data to cells |
| `format-range` | Apply formatting |
| `find` / `replace` | Search and update |
| `copy` | Duplicate data |
| `clear-contents` | Remove values only |
| `clear-all` | Remove values + formatting |
